LinkShrink

A link shortening service accessible with cURL.

Try It Out

curl -X POST https://reznicek.dev/linkshrink \
     -H "Content-Type: application/json" \
     -d "{\"decoded\": \"https://www.google.com\"}"

When you shorten a URL with LinkShrink, you'll be provided with an 8-character identifier. Simply concatenate this identifier to the domain and send a GET request to access the link.

This example request will redirect you to Google!

curl -X GET https://reznicek.dev/linkshrink/d2abbfe2 -L

How It Works

Make sure the link is valid; that it fits all HTTP standards and is the correct format.

Check if the link has already been saved before.

Generate a random unique identifier for the link. Ensure this new identifier doesn't accidentally already exist before continuing.

Save the match in a database, and then return it to the user. The link is ready to be used!

This is a Proof-of-Concept SpringBoot application meant only for educational and not serious production purposes. You can find more information about it on my GitHub here.
